When to use this component

Use this component as an alternative to the select component when the user has to choose from a large list of options.

When not to use this component

Before using the autocomplete component, try asking users questions which will allow you to present them with fewer options.

Asking questions means youâ€™re less likely to need to use the autocomplete component, and can consider using a different solution, such as radios.

How it works

As the user types, the list of options will be reduced depending on what they type.

For example, if the user types 'AL' when using this component to select a list of American states, it would present Alabama and Alaska in the dropdown.
