When to use this component

Use this component when the user needs to be aware that the link will be going to an external URL.

When not to use this component

Do not use this component when the link does not take the user to an external URL.

How it works

To show visually that a url is external, apply the class: .notices to make the external icon appear.

The icon is applied via css and does not require any additional HTML. Ensure the background image URL is correct otherwise the svg icon will not appear.

There is also hidden assistive text to tell screenreaders this is an notices as the icon will no be visible.
