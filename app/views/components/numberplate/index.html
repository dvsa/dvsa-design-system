Use the Numberplate display component to show the numberplate of a vehicle.

When to use this component

Use this component when it is important for the user to enter the correct email address.

When not to use this component

Do not use this component to repeat other inputs, such as entering a first name, as the email address error functionality will not work.

How it works

As the user enters their email address it will be repeated to them in the inset text component.

This is to aid the user in entering the correct address, by giving them an opportunity to notice errors in what they have typed.

The component also detects if the user may have typed something wrong, by cross checking what they have entered with common mistakes via AJAX.

If the component does detect a possible error, it will show an error message to the user with what is typed currently and a suggestion.
