When to use this pattern

Use this pattern when users need to answer a simple question. This pattern can be used in multiple ways, such as being part of a series of questions.

How it works

The user selects an answer to the question. Then selects the action buttons to continue their journey. The details component is included in this pattern as an option to give more details about the question to the user.

The action might take the user further in a journey. Or if the action can be performed immediately without further questions, show a success message at the top of the page.
